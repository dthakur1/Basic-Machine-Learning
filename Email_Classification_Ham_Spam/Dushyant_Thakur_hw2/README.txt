Once inside the directory of this homework folder, type the following command:

$./hw2 <test directory>

clearly in the source code and it is commented out.

After execution, this will create a result.txt file which contains the accuracies.

First line represents the accuracy in spam folder of test directory with and without stop words.
First line represents the accuracy in ham folder of test directory with and without stop words.
